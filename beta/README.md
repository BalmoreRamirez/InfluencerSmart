# Influencer Smart App

Base inicial con Vue 3 + Vite + TypeScript + PrimeVue + Pinia + Vue Router + Axios + ESLint + Prettier + VueUse + Tailwind CSS.

## Requisitos

- Node.js 18+ (recomendado)
- npm 9+

## Uso rapido

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: servidor de desarrollo
- `npm run build`: build de produccion
- `npm run preview`: previsualizar build
- `npm run lint`: lint con ESLint
- `npm run format`: formateo con Prettier

## Estructura

- `src/main.ts`: entry point
- `src/router`: rutas
- `src/stores`: estado global (Pinia)
- `src/services/http.ts`: cliente Axios base
- `src/views`: vistas principales
